About the project

OpenBikeSharing is free software (as in freedom) and is distributed under the GPLv3 license. This means you are free to use, modify and share it. You are also encouraged to contribute any piece of code, report potential bugs or simply make any suggestions to improve it.

Data are provided by the CityBikes API.

OpenBikeSharing uses software libraries like osmdroid and osmbonuspack.

The source code is currently hosted on GitHub.
